Modifica-mos o projeto focando para uso em celulares, usando a o metodo de mobili-first no CSS para garantir melhor funcionamento em telas pequenas. Também adicionando contraste do botão principal para facilitar o uso e os botões de ação foram movidos para baixo do site, deixando mais fácil de alcançar no celular. Ajustamos o tamanho dos botões e os espaçamentos para evitar cliques errados e criamos um popup para o formulário, ajudando a organizar as informações e não ficar a tela cheia.
Também realizamos a componentização do projeto para separar as responsabilidades or parte e facilitando a leitura do código, manutenção e permitindo reutilização do código.
